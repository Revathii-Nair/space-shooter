# Space Shooter

A simple 2D arcade space shooter built with Godot 4. Survive as long as possible by shooting down asteroids and avoiding collisions!

This game was made to learn and understand how the Godot Engine works.

## Play Online

[Play Space Shooter in your Browser Here.](https://your-vercel-deployment-url)

## Features

- **Endless Gameplay:** Meteors spawn randomly with varying speeds and sizes.
- **Score System:** Earn 5 points for every meteor you destroy.
- **Health Tracking:** You start with 3 lives. Colliding with a meteor loses a life.

## Controls

- **Move:** `W`, `A`, `S`, `D` or `Arrow Keys`.
- **Shoot:** `Spacebar` or `Left Mouse Click`.

## Tech Stack

- **Engine:** Godot 4
- **Language:** GDScript
- **Deployment:** Vercel (Web Build)
